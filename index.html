Freelancer هو قالب Bootstrap مجاني تم إنشاؤه بواسطة Start Bootstrap. يتضمن التحميل ملفات المصدر الكاملة بما في ذلك HTML و CSS و JavaScript بالإضافة إلى أوراق أنماط SASS الاختيارية لسهولة التخصيص.
